# Beans - 원두 판매 쇼핑몰

## 프로젝트 개요
원두 판매 상세 페이지. 스페셜티 커피 감성의 미니멀 프리미엄 디자인.

## 기술 스택
- HTML / CSS / JavaScript (Vanilla)
- 별도 프레임워크 없음

## 파일 구조
```
beans/
├── CLAUDE.md
├── design.md
├── index.html        ← 상품 상세 페이지
├── css/
│   └── style.css
└── js/
    └── main.js
```

## MVP 범위 (1차)
### 상품 정보 구역
- 상품 이미지 (메인 + 썸네일)
- 상품명
- 가격
- 원산지, 로스팅 레벨, 플레이버 노트
- 재고 상태

### 구매 옵션 구역
- 용량 선택 (200g / 500g / 1kg)
- 분쇄 옵션 (홀빈 / 에스프레소 / 드립)
- 수량 선택
- 장바구니 / 바로구매 버튼

## 추후 추가 예정
- 상품 설명 구역 (원두 스토리, 추출 가이드, 리뷰)
- 세트 구성 기능
- 정기 구독 옵션
